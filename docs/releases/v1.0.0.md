# v1.0.0 · 2026-07-06 · First Public Release

> 🎉 **首个公开版本**。完整短剧平台 MVP：后端 + 管理后台 + 移动端 + 互动节点 + TOS 上传。

## ✨ Features

### 后端 (`ruoyi-fastapi-backend/`)

- FastAPI + SQLAlchemy 2.0 (async) + Pydantic v2
- MySQL 5.7+ / PostgreSQL 12+ 双兼容
- Redis 6.2+ 缓存与限流
- OAuth2 + JWT 双 token 机制
- 短剧业务：`module_drama/`
  - Drama / DramaNode / DramaEpisode / DramaChoice
  - Feed 流、点赞收藏、评论、互动节点编排
  - TOS / S3 兼容预签名分片上传 + 断点续传
- RuoYi 后台能力继承：用户/角色/菜单/部门/字典/参数/日志/定时任务/监控/代码生成/AI 对话
- Alembic 数据库迁移（`250515_drama` → `250516_drama_align_v2`）
- `ruoyi` CLI：`app run | app doctor | app routes | db upgrade | cache clear | dev lint`

### 管理后台 (`ruoyi-fastapi-frontend/`)

- Vue 3 + Vite + Element Plus + Pinia + Vue Router 4
- 短剧 CRUD、节点编辑、剧集管理、用户管理、运营位
- API 路径别名兼容：`/api/app/*` ↔ `/api/*`
- 默认端口：5188

### 移动端 (`mobile-app/`)

- uni-app + Vue 3 + Vite + Tailwind CSS
- H5 / 微信小程序 / 抖音小程序 / 多端编译
- 默认 API 前缀：`VITE_APP_API_PREFIX=/api/app`，登录 `/api/auth/*`

### 工程化

- Docker Compose（MySQL / PostgreSQL / Drama 编排）
- Nginx 反向代理示例
- 14 节 README + 架构图 + Troubleshooting 矩阵 + 验收清单
- Alembic 迁移链路完整
- GitHub Issue 模板（Bug / Feature / Question）
- GitHub PR 模板 + Conventional Commits

## 📦 Installation

```bash
git clone https://github.com/flowerjunjie/ai-interactive-drama.git
cd ai-interactive-drama

# 后端
cd ruoyi-fastapi-backend
pip install -r requirements.txt
cp .env.example .env.dev   # 改 DB / Redis 配置
ruoyi app run --env=dev    # http://localhost:19199/docs

# 管理后台
cd ../ruoyi-fastapi-frontend
npm install --registry=https://registry.npmmirror.com
npm run dev                # http://localhost:5188  admin / admin123

# 移动端 H5
cd ../mobile-app
pnpm install
pnpm dev:h5
```

详见 [README](../../README.md)。

## 🔐 Security

- 默认 `admin / admin123` **必须**在生产环境修改
- `.env.prod` 中 `JWT_SECRETKEY` **必须**改为强随机
- TOS / S3 凭证走环境变量，**不要**入库
