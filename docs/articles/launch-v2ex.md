## 正文

```
最近 5 个月我们团队搭了一个 AI 互动短剧平台，后台 + C 端 + 移动端全套。
现在完整开源了，MIT 协议，可以商用。

项目地址：https://github.com/flowerjunjie/ai-interactive-drama

核心特性：
- 后端：FastAPI + SQLAlchemy 2.0 async + MySQL/PG + Redis
- 后台：Vue3 + Element Plus + Vite
- 移动端：uni-app + Tailwind（H5 / 微信小程序 / 多端）
- 短剧业务：Feed 流、互动节点（分支剧情）、点赞收藏、TOS 分片上传
- 鉴权：OAuth2 + JWT
- 工程化：CLI、ruoyi app doctor、Alembic、Docker Compose

技术选型上没用 Java/Spring Cloud 那套重型的，因为我们团队只有 4 个人，
Python + Vue 迭代速度明显快一截。

应用场景：短剧 C 端产品、互动剧情创作工具、内容审核工作流。

附几张仓库内信息：
- 14 节 README，5 分钟能跑起来
- Alembic 迁移、CLI 工具、Troubleshooting 矩阵
- GitHub Actions / Docker / Nginx 配置样例

商业上想联运 / 软著联名 / 行业合作的可以 issue 联系，
技术问题开 Discussion 比较合适。

欢迎提 Issue / PR / Star ✨
```
