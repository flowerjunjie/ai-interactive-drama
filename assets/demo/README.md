## 录制工具建议

- **macOS**：QuickTime Player / Kap（开源）
- **Windows**：ScreenToGif / OBS
- **Linux**：Peek / OBS
- **B 站视频**：OBS + 旁白 + 字幕

## GIF 导出建议

- 帧率：15-24 fps
- 分辨率：管理后台 1280px 宽 / 移动端 375px 宽
- 时长：单条 30-60 秒为佳
- 大小：< 10 MB（README 引用别超 5 MB）
